°°📚 BookWorld: Sistema de Gestión Bibliotecaria

**BookWorld** es una solución de software de escritorio desarrollada en **C# (.NET 6.0)** como parte del **Taller de Programación (Semana 6)**. El sistema automatiza el flujo de préstamos, devoluciones y gestión de inventario, eliminando la necesidad de registros manuales y asegurando la integridad de las reglas de negocio.

°°  Características Principales
- **Gestión de Entidades**: Registro y visualización de Estudiantes y Catálogo de Libros.
- **Reglas de Negocio Estrictas**:
    - Límite máximo de **3 libros simultáneos** por usuario.
    - Control de **disponibilidad en tiempo real** (un libro prestado no puede asignarse nuevamente).
- **Arquitectura Robusta**: Uso de Programación Orientada a Objetos (POO) con Herencia, Encapsulamiento y Polimorfismo.

°°  Stack Tecnológico
- **Lenguaje:** C#
- **Framework:** .NET 6.0
- **Entorno:** Visual Studio 2022
- **Patrón de Diseño:** Singleton (Instancia única para la persistencia de datos en memoria).
